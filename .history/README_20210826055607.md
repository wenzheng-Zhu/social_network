# README

说明：
  该小程序比较简单，只是根据用户关注，用户发布信息，用户查询所关注的用户发布的信息这三个需求简单的写了下框架和接口，接口也比较简单，没有用grape,或者jbuilder相关gem, 是一个纯简单的接口项目，不包含测试，不包含分页；
  
  



1. ruby版本：2.6.5， rails版本6.1.4.1
2. 数据库： mysql 
3. users表， articles表， relationships表
4. bundle,rails db:create, rails db:migrate后，可以运行 rails db:seeds创建测试数据
