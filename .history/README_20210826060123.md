# README

一、说明：
1. 写的比较简单，主要是根据三个只是根据用户关注，用户发布信息，用户查询所关注的用户发布的信息这三个需求简单的写了下框架和接口，接口也比较简单，没有用grape,或者jbuilder相关gem, 是一个纯简单的接口项目，不包含测试，不包 含分页。
2. ruby版本：2.6.5， rails版本6.1.4.1
3. 数据库： mysql 
4. 总共三张表：users表， articles表， relationships表
5. bundle,rails db:create, rails db:migrate后，可以运行 rails db:seeds创建测试数据
